一键 extroot（整盘 overlay 到 U 盘）

作用：把当前可写层 /overlay 迁移到已连接的 USB 磁盘分区，重启后所有软件包与配置默认写入 U 盘，相当于官方文档中的 extroot。 与上方「仅 opkg 备用目标」不同，这是系统级迁移。

风险：操作错误可能导致无法启动。请先备份配置；仅选择U 盘分区（本向导仅允许 /dev/sd* 分区，避免误选内置 eMMC）。 勾选「格式化」将清空该分区全部数据。

多分区 U 盘：swap 与数据盘（本向导不自动处理）

	

		建议在 PC（如 Ubuntu 磁盘工具）将 U 盘设为 GPT，例如：sda1 专用于 overlay（上方一键 extroot）、sda2 作 swap、sda3 作日常数据。分区请在插入路由器之前完成；仅第一分区由本向导格式化为 ext4（若勾选格式化）。
	

	

		重启生效后，到 系统 → 挂载点 继续添加：sda2 设为交换分区；sda3 挂载到如 /mnt/data（ext4 需先 mkfs.ext4，可在 SSH 或 PC 上完成）。
	

	
# SSH 参考：swap（替换为 blkid 查到的 UUID）
opkg install block-mount kmod-fs-ext4
mkswap /dev/sda2
uci set fstab.swap=swap
uci set fstab.swap.uuid='YOUR-SDA2-UUID'
uci set fstab.swap.enabled='1'
uci commit fstab
/etc/init.d/fstab enable
/etc/init.d/fstab restart

# 数据分区挂载到 /mnt/data
mkdir -p /mnt/data
uci set fstab.data=mount
uci set fstab.data.target='/mnt/data'
uci set fstab.data.uuid='YOUR-SDA3-UUID'
uci set fstab.data.fstype='ext4'
uci set fstab.data.enabled='1'
uci commit fstab
/etc/init.d/fstab restart






	
安装到 U 盘（opkg 备用目标）

	

		说明：系统自带的「软件包」页面默认把包装进根文件系统（overlay）。
		若希望把单个软件装到已挂载的 U 盘分区，可在此配置 dest arxusb，并在下方使用「安装包」或 SSH 执行
		opkg install -d arxusb <包名>。
		装到 U 盘的内容位于该分区下的 usr/ 等目录，不会自动合并进当前运行根路径，适合节省内置闪存或做隔离；若要让所有插件默认都装到 U 盘，请使用 OpenWrt 的 extroot / 外置 overlay（将 /overlay 迁到 U 盘）。
